#### Описание:
Приложение для формирования списка задач. Есть возможность добавлять, изменять, удалять задачи.
Менять статус задачи. Имеется фильтр для отображения невыполненных задач.

#### Используемые технологии:
- Spring MVC
- Spring Security
- Spring Data JPA
- Spring AOP  
- PostgreSQL
- Liquibase
- Log4j
- CheckStyle  
- Thymeleaf
- HTML
- CSS
